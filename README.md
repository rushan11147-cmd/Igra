# Horror Factory

Психологический хоррор + симулятор производства. **Godot 4.7**.

## Уникальные механики

### Умный монстр
- Запоминает укрытия и проверяет их
- Слышит шум, реагирует на фонарик
- Отключает свет, ломает оборудование
- Каждое прохождение отличается (`MonsterMemory`)

### Правила завода (вирусный хук)
Каждую ночь — новые правила. Нарушение = угроза + потеря рассудка.
> *«Никогда не открывай дверь цеха №8 после 03:00»*

С ночи 4 в **03:00** — стук. Голос. Выбор: `[1]` открыть · `[2]` уйти · `[3]` отключить питание · `[4]` заглянуть

### Живой завод
Между ночами: заваренные двери, новые коридоры, лифт на не тот этаж (`LivingFactory`)

### Рассудок
Голоса, ложные монстры, неверные показания приборов, искажённые задания (`SanitySystem`)

### 4 типа существ
| Тип | Поведение |
|-----|-----------|
| Наблюдатель | Смотрит, не атакует, реагирует на свет |
| Охотник | Реагирует на шум |
| Ползун | Только вентиляция |
| Тень | Только в темноте |

### Случайные события
Пожар, утечка газа, сирена, звонок телефона — каждая ночь уникальна

### Выбор в конце смены
- Выполнить план (+деньги)
- Остановить производство (↓ угроза)
- Исследовать завод (документы)

### Напарник по рации
Дмитрий → тишина → чужой голос

### 7 концовок
Сбежать · Уничтожить завод · Спасти сотрудников · Продать секреты · Стать субъектом · Плохая · Секретная

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD / Shift / Ctrl | Движение / бег / присед |
| Z / X | Выглядывание |
| E | Взаимодействие |
| F | Фонарик |
| T | Ответить по рации |
| Tab | Планшет |
| 1–4 | Выбор при событии цеха №8 |
